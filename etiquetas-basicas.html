Etiquetas básicas HTML

  • !DOCTYPE html: La declaración de tipo de documento HTML, también conocida como DOCTYPE, es la primera línea de código requerida en todo documento HTML o XHTML. Esto asegura que la página web sea analizada de la misma manera en diferentes navegadores. Utilizaremos el DOCTYPE para HTML5, que es compatible con todos los navegadores.
  • <html>: Es la etiqueta en la que anidaremos todo nuestro documento. Representa la raíz de un documento HTML. Aunque puede ser implícita en HTML, es requerida para abrir y cerrar en XHTML.
  • <head>: Se utiliza para incluir información sobre la página que no se muestra directamente. Dentro de esta etiqueta puedes colocar el título de la página, enlaces a archivos de estilo (CSS) y scripts (JavaScript).
  • <body>: Contiene todo el contenido visible de la página web, incluyendo títulos, párrafos, listas y otros elementos.
  • <h1, h2, h3...h6>: Estas etiquetas definen títulos y subtítulos en una página web, organizando el contenido en una jerarquía. Utilizarlas correctamente ayuda a estructurar el contenido y mejora la optimización en motores de búsqueda (SEO).
  • <p>: Se utiliza para definir un párrafo en HTML. Los párrafos se muestran en bloques de texto separados, lo que ayuda a estructurar el contenido de manera legible.
  • <a>: Se utiliza para crear enlaces en HTML. Puedes usar el atributo href para especificar la URL del destino. También puedes usar el atributo target para definir cómo abrir el enlace.

      Atributos comunes:

    • href: Define la URL del destino del enlace.
    • target: Especifica dónde abrir el enlace (por ejemplo, _blank para abrir en una nueva pestaña).
    • title: Proporciona información adicional que aparece como un tooltip cuando el usuario pasa el cursor sobre el enlace.
  • <header>: Representa un grupo de ayudas introductorias o de navegación. Puede contener elementos de encabezado, un logo y un formulario de búsqueda.
  • <nav>: Representa una sección de navegación, proporcionando enlaces de navegación dentro del documento o a otros documentos.
  • <section>: Representa una sección genérica independiente de un documento. Las secciones siempre deben tener un título.
  • <img>: Se utiliza para insertar imágenes en la página. Ejemplo: <img src="imagen.jpg" alt="Descripción de la imagen">
  • <ul>: Se utiliza para crear listas desordenadas. Ejemplo: <ul><li>Elemento 1</li></ul>
  • <ol>: Se utiliza para crear listas ordenadas. Ejemplo: <ol><li>Elemento 1</li></ol>
  • <table>: Se utiliza para crear tablas en HTML. Ejemplo: <table><tr><td>Celda 1</td></tr></table>
  • <form>: Se utiliza para crear formularios interactivos. Ejemplo: <form action="tu_accion"></form>
