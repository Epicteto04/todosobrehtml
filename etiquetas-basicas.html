Etiquetas básicas HTML

  • !DOCTYPE html>: La declaración de tipo de documento HTML, también conocida como DOCTYPE, es la primera línea de código requerida en todo documento HTML o XHTML. La declaración DOCTYPE es una instrucción al navegador web sobre la versión de HTML en la cual está escrita la página. Esto asegura que la página web sea analizada de la misma manera en diferentes navegadores web. En nuestro caso vamos a utilizar el DOCTYPE que se utiliza para HTML5 y que va a ser compatible con todos los navegadores web. Es importante saber que sólamente tiene etiqueta de apertura
  • <html>: Es la etiqueta en la que anidaremos todo nuestro documento. El elemento HTML (o elemento HTML raiz) representa la raiz de un documento HTML. El resto de elementos descienden de este elemento. Dado que el elemento es el primero en un documento, aparte de los comentarios, se llama el elemento raíz. A pesar de que esta etiqueta puede ser implicita, o no requerida en HTML, sí es requerida para abrir y cerrar en XHTML.
  • <head>: La etiqueta head se utiliza para incluir información sobre la página que no se muestra directamente en el contenido visible. Dentro de la etiqueta <head> puedes colocar elementos como el título de la página, enlaces a archivos de estilo (CSS), scripts (JavaScript) y metadatos que ayudan a definir cómo se presenta y se comporta la página en el navegador.
  • <body>: La etiqueta body contiene todo el contenido visible de una página web, incluyendo títulos, subtítulos, párrafos, listas, tablas y otros elementos que los usuarios verán en el navegador. Todo el contenido que aparece en la página se coloca dentro de esta etiqueta.
  • <h1, h2, h3...h6>: Las etiquetas de título h1, h2, h3, hasta h6, se utilizan para definir títulos y subtítulos en una página web. Estas etiquetas organizan el contenido en una jerarquía, donde h1 representa el título más importante y h6 el de menor relevancia. Utilizar correctamente estas etiquetas ayuda a estructurar el contenido de manera clara y accesible, y también mejora la optimización en motores de búsqueda (SEO).
  • <p>: La etiqueta <p> se utiliza para definir un párrafo en HTML. Los párrafos se muestran en bloques de texto separados, con márgenes automáticos en la parte superior e inferior para una adecuada separación entre ellos. Esta etiqueta ayuda a estructurar el contenido de manera legible y organizada.
  • <a>: La etiqueta <a> se utiliza para crear enlaces en HTML. Se conoce como el "elemento de anclaje". Puedes usar el atributo href para especificar la URL del destino del enlace. Además, puedes usar el atributo target para definir cómo se debe abrir el enlace (por ejemplo, en una nueva pestaña). Esta etiqueta también puede ser utilizada para enlaces a otras secciones de la misma página utilizando el atributo id y el signo de número (#).

      Atributos comunes:

    • href: Define la URL del destino del enlace.
    • target: Especifica dónde abrir el enlace (por ejemplo, _blank para abrir en una nueva pestaña).
    • title: Proporciona información adicional que aparece como un tooltip cuando el usuario pasa el cursor sobre el enlace.
  • <header>: El elemento de HTML Header (<header>) representa un grupo de ayudas introductorias o de navegación. Puede contener algunos elementos de encabezado, así como también un logo, un formulario de búsqueda, un nombre de autor y otros componentes.
  • <nav>: El elemento de sección de navegación: El elemento HTML <nav> representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos. Ejemplos comunes de secciones de navegación son menús, tablas de contenido e índices.
  • <section>: El elemento de sección genérica:El elemento de HTML <section> representa una sección genérica independiente de un documento, que no tiene un elemento semántico más específico para representarla. Las secciones siempre deben tener un título, con muy pocas excepciones.
