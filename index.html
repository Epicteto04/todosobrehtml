Esta web está diseñada para fomentar un aprendizaje proactivo en el desarrollo web. Se recomienda realizar búsquedas en fuentes fiables como Mozilla, el W3C, y las bases de datos de Microsoft o Google, especialmente en lo que respecta a cursos de programación. Al abordar nuevos conceptos, es beneficioso contrastar la información obtenida con diversos recursos, lo que enriquecerá tu comprensión y habilidades. Para encontrar información de calidad, considera utilizar motores de búsqueda académicos como Google Scholar, donde puedes acceder a artículos revisados por pares. También puedes explorar plataformas de educación en línea como Coursera o edX, que ofrecen cursos impartidos por universidades reconocidas. Foros y comunidades como Stack Overflow y Reddit pueden ser útiles para resolver dudas y ver cómo otros desarrolladores abordan problemas similares. Además, asegúrate de verificar la fecha de publicación de los recursos que consultes, ya que el mundo del desarrollo web evoluciona rápidamente y es crucial trabajar con información actualizada. No dudes en explorar diferentes enfoques y soluciones para fortalecer tu aprendizaje de manera efectiva. Combinar la investigación con la práctica y la reflexión crítica te permitirá construir una base sólida en el desarrollo web.

Primeros pasos para construir nuestra primera web

Aquí aplicaré técnicas básicas de escritura en HTML para aprender las mejores prácticas y aplicarlas en una web final

Lo primero que tenemos que hacer es elegir nuestro entorno para empezar a picar código HTML, elige lo que te haga sentir más cómodo, desde el bloc de notas que viene por defecto en tu sistema (Windows o Mac) hasta entornos más específicos como pueden ser Visual Studio Code (es el que yo he escogido, ya que en un futuro podrá servirme para aprender otros lenguajes y ya me he acostumbrado), Notepad++ o Atom.

Una vez que hemos elegido la aplicación desde la que vamos a trabajar, nos interesa saber cuáles son las etiquetas principales que deberemos de utilizar para construir una web de la manera más sencilla posible. En esta ocasión he querido construir una Web que esté dedicada a HTML y a guiar a usuarios novatos en sus primeros pasos. Continúa leyendo la sección Etiquetas Básicas

Una vez que os familiaricéis con estas etiquetas, ya podréis construir una web básica con HTML, os recomiendo utilizar tarjetas de estudio tipo Anki para repasarlas.
