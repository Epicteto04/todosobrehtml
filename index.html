Primeros pasos para construir nuestra primera web

Aquí aplicaré técnicas básicas de escritura en HTML para aprender las mejores prácticas y aplicarlas en una web final

Lo primero que tenemos que hacer es elegir nuestro entorno para empezar a picar código HTML, elige lo que te haga sentir más cómodo, desde el bloc de notas que viene por defecto en tu sistema (Windows o Mac) hasta entornos más específicos como pueden ser Visual Studio Code (es el que yo he escogido, ya que en un futuro podrá servirme para aprender otros lenguajes y ya me he acostumbrado), Notepad++ o Atom.

Una vez que hemos elegido la aplicación desde la que vamos a trabajar, nos interesa saber cuáles son las etiquetas principales que deberemos de utilizar para construir una web de la manera más sencilla posible. En esta ocasión he querido construir una Web que esté dedicada a HTML y a guiar a usuarios novatos en sus primeros pasos. Continúa leyendo la sección Etiquetas Básicas

Una vez que os familiaricéis con estas etiquetas, ya podréis construir una web básica con HTML, os recomiendo utilizar tarjetas de estudio tipo Anki para repasarlas.
